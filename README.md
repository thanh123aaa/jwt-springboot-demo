# Bài tập Spring Boot

## Câu 1: Spring Boot cơ bản
- Tạo project Spring Boot
- Tạo REST API đơn giản
- Kiểm tra API chạy tại trình duyệt / Postman

## Câu 2: JWT + Spring Boot
- Tạo JWT token bằng thư viện jjwt
- Gửi token qua Header Authorization
- Xác thực token khi gọi API

### API sử dụng
- GET /token  
  → Trả về JWT token

- GET /hello  
  Header: Authorization: Bearer <token>

## Công nghệ
- Java
- Spring Boot
- JWT (jjwt)
- Maven
